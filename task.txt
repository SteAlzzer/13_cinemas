Что в кино   задача #13
Порой хочется узнать самые классные фильмы, идущие в данный момент, однако заходя на афишу(http://www.afisha.ru/msk/schedule_cinema/) трудно определить, у каких из них рейтинг выше (например, рейтинг на Кинопоиске).

Скрипт должен выводить самые популярные фильмы, идущие в данный момент, отсортированные по рейтингу.
Для этого необходимо:

Получить список фильмов отсюда(http://www.afisha.ru/msk/schedule_cinema/), и заодно вытащить количество кинотеатров, в котором идет фильм (а то артхаус тоже часто высоко оценивают)
Для каждого фильма получить среднюю оценку, а также количество оценок. Для этого стоит использовать кинопоиск(https://www.kinopoisk.ru/)
Отсортировать фильмы по рейтингу. Можно добавить дополнительный параметр, который будет показывать только фильмы, идущие в большом количестве кинотеатров (а значит, для них проще найти сеанс).
Вывести первые 10 фильмов
Для парсинга страниц рекомендуется использовать beautifulsoup4(https://pypi.python.org/pypi/beautifulsoup4)
А для получения страниц - модуль requests(http://docs.python-requests.org/en/master/)
